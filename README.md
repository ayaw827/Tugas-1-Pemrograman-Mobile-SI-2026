## 🛍️ 1. Halaman Katalog (Album Taste)

Di halaman utama ini, user bisa melihat daftar produk album.

**Fitur:**

* 📦 **List produk** (3 versi album: Savory, Full Spread, Tin Case)
* 🖼️ **Gambar produk**
* 📝 **Deskripsi singkat** (isi album)
* 💰 **Harga produk**
* ➕ **Tombol “Add”**

  * Untuk menambahkan produk ke keranjang

## 🛒 2. Keranjang Kosong (Empty Cart State)

Saat belum ada barang:

**Fitur:**

* 🛒 Icon + teks **“Your cart is empty”**
* 🔁 Tombol **“Continue Shopping”**

  * Balik ke halaman katalog

## 🧺 3. Keranjang Belanja (Shopping Cart)

Setelah user menambahkan produk:

**Fitur utama:**

* 📋 **List item dalam keranjang**

  * Nama produk
  * Harga
  * Gambar
* ➖➕ **Quantity control**

  * Tambah atau kurangi jumlah barang
* 🗑️ **Delete item**

  * Hapus produk dari cart
* 💵 **Subtotal per item**
* 🧮 **Total harga keseluruhan**
* ✅ Tombol **Checkout**

## 💳 4. Checkout Modal

Saat klik checkout:

**Fitur:**

* 🧾 Ringkasan:

  * Total harga
  * Jumlah item
* ❌ Tombol **Cancel**
* ✔️ Tombol **Confirm**

## 🎉 5. Konfirmasi Pesanan

Setelah klik confirm:

**Fitur:**

* 🔔 Notifikasi **“Order placed!”**
* 🔄 Biasanya:

  * Cart dikosongkan
  * Kembali ke halaman katalog


## 🔁 Flow Penggunaan (Singkat)

1. User lihat produk → klik **Add**
2. Masuk ke **Shopping Cart**
3. Atur jumlah / hapus item
4. Klik **Checkout**
5. Konfirmasi → **Order placed**
